# Prompt — From Frog Image to Weekly Ritual

Reconstructed prompt for [`index.html`](./index.html) — the original wasn't
preserved, so this prompt is written to replicate the sample.

---

Write a short research brief as a polished, shareable web page (one
self-contained HTML file) titled "From Frog Image to Weekly Ritual".

Subject: how a blank frog meme image became a recurring internet ritual —
trace the origin, explain why the cadence (not the novelty) drives the
ritual, and what decision-makers should and shouldn't conclude from it.

Structure:
- Executive summary ("What decision-makers need to know")
- Origin analysis ("Call it Reddit-native, not Reddit-originated")
- How it became a ceremony; why cadence beats novelty
- "What not to overclaim" — honest caveats section
- Origin → energy → ritual model
- A validation checklist

Design: editorial report layout — clear typographic hierarchy, generous
whitespace, callout boxes for the checklist and caveats. Looks like a paid
analyst note, not a blog post. Embedded CSS only, no external dependencies,
mobile responsive.
