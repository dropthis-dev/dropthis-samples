# Prompt — Get Early Access (multi-step form)

Reconstructed prompt for [`index.html`](./index.html) — the original wasn't
preserved, so this prompt is written to replicate the sample.

---

Create a polished multi-step signup form ("Get Early Access") in one
self-contained HTML file.

- 4 steps, one question per screen: name → contact email → about your
  project → how did you hear about us. Finish with a "You're on the list!"
  success screen.
- Animated progress bar, back/next navigation, Enter-to-advance.
- Client-side validation per step (required fields, email format) with
  inline error states.
- All state kept in JS; smooth slide transitions between steps.
- Modern, friendly visual design; mobile responsive; embedded CSS/JS, no
  external dependencies.
