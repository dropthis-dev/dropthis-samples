# AI-Native Slide Systems — demo decks

Two keyboard-driven, single-file HTML slide decks:

- `index.html` — **AI-Native Slide Systems**, a demo deck about building slides as plain HTML.
- `mcp-is-dead.html` — **"MCP is dead." — Is it though?**, a longer opinion deck built the same way.

Open either file directly in a browser. Arrow keys navigate.
