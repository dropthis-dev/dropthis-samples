# Shine Brown — TikTok swipe deck

Mobile-first swipe deck driving to the BYROKKO Shine Brown product page.
Unlike the single-file samples this one is a small multi-file package:

```
index.html      shell that renders the deck
app.js          swipe/render logic
content.json    slide copy (validated by content.schema.json)
blueprint.json  deck blueprint (tiktok@1.0.0)
theme.json      brand tokens (+ theme.css / styles.css)
assets/         slide artwork
brief.md        the original creative brief
```

Open `index.html` in a browser, or publish the whole folder as one drop.
