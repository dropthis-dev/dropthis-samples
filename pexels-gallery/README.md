# Field Notes — Pexels photo wall

Masonry photo gallery in one HTML file: 18 photos from the Pexels CDN,
lazy-loaded with staggered scroll-in reveals, hover captions, and a native
`<dialog>` lightbox (click to zoom, Esc to close).

Open `index.html` in a browser. Images load from images.pexels.com
([Pexels license](https://www.pexels.com/license/)), so the file itself
stays tiny.
