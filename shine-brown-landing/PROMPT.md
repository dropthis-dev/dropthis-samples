# Prompt — Shine Brown landing page

Reconstructed prompt for [`index.html`](./index.html) — the original wasn't
preserved, so this prompt is written to replicate the sample.

---

Create a single-file product landing page for BYROKKO Shine Brown — a
natural tanning oil that accelerates and deepens tanning.

- Hero with product name, benefit-led headline ("faster, deeper results"),
  and primary CTA to the official product page
  (https://www.byrokko.com/global/products/shine-brown).
- Sections: how it works, key benefits, ingredients/clean formula, social
  proof, FAQ, closing CTA.
- Warm sun-drenched palette matching a tanning brand; bold display
  typography; mobile-first responsive layout.
- One self-contained HTML file, embedded CSS/JS, no external dependencies.
- SEO basics: proper title, meta description, semantic headings.
