# Prompt — SPA Calculator

Reconstructed prompt for [`index.html`](./index.html) — the original wasn't
preserved, so this prompt is written to replicate the sample.

---

Create a minimal single-page calculator app in one self-contained HTML file.

- Standard four-function calculator (+, −, ×, ÷) with decimal support,
  clear, and backspace.
- Clean card-style layout centered on the page; large display, tactile
  button grid, keyboard input support.
- No frameworks, no external dependencies — embedded CSS and JS only.
- Title it "SPA Calculator".
