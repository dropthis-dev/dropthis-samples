# dropthis-examples

Small, self-contained examples and demos built with / for [dropthis](https://dropthis.app) —
the publish layer between AI and the internet.

## Examples

| Folder | What it is |
| --- | --- |
| [`frogger/`](./frogger) | **Publish Frog** — an 8-bit Frogger parody about the pain of getting an AI-generated file online. Single HTML file, no build step, no dependencies. |

## Running

Every example is self-contained. Open the HTML file directly in a browser, or drop it online with dropthis.
